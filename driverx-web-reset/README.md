# DriverX Web Landing

Página web pública estática de DriverX.

## Cómo correrla

1. Abre la carpeta `driverx-web-reset` en Visual Studio Code.
2. Abre `index.html`.
3. Clic derecho > `Open with Live Server`.

## Archivos

- `index.html`: contenido de la página.
- `styles.css`: diseño, colores y responsive.
- `script.js`: menú móvil, animaciones y formulario demo.

## Colores principales

- Negro premium: `#050607`
- Charcoal: `#090b0f`
- Rojo DriverX: `#ff172b`
